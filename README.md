# How to run it
- Git clone du projet
- Compilation: 
    `npm i --legacy-peer-deps`
- Démarrage serveur JSON: 
    `json-server --watch .\json-server\data.json --port 3000`
- Démarrage Front-End: 
    `ng serve --port 4200`
- Ouvrir son navigateur à l'adresse: 
    `http://localhost:4200/`
